Repo Graveyard v0.0.1 - Portable Version

This is a portable version of Repo Graveyard. No installation required!

How to use:
1. Double-click "Repo Graveyard.exe" to start the application
2. First time users will need to enter their AI API Key in Settings
3. The app stores your settings locally

Features:
- Browse and bury dead GitHub repositories
- AI-generated eulogies (Gemini, OpenAI, DeepSeek)
- Undertaker identity and leaderboard system
- Offline capable (requires internet for GitHub/Supabase access)

System Requirements:
- Windows 10/11 (64-bit)
- WebView2 Runtime (usually pre-installed on Windows 11)

License:
- This project is open source
- Visit: https://github.com/youyouhe/Repo-Graveyard

Note:
- This is a portable version - all files are contained in the .exe
- You can delete the folder to uninstall
