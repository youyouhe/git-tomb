# Portable Release for Repo Graveyard v0.0.1

## Files

- `Repo Graveyard.exe` - Main executable (5.5MB)
- `README.txt` - Usage instructions
- `README.md` - Project documentation

## Usage

1. Extract the folder
2. Double-click `Repo Graveyard.exe` to start
3. First launch: Enter your AI API Key in Settings
4. Start using!

## System Requirements

- Windows 10/11 (64-bit)
- WebView2 Runtime (usually pre-installed on Windows 11)
- Internet connection (for GitHub/Supabase access)

## Uninstall

Simply delete the portable folder. No registry entries or system files are created.

## Notes

- All application data is stored in: `%LOCALAPPDATA%\com.repograveyard.app\`
- AI API Keys are stored locally and never sent to our servers
- This is a standalone portable version - no installation required
